## 🎯 Prompt: Create Basic Admin Panel UI (HTML + CSS)

Design and develop a **responsive Admin Panel UI** for a "Wedding Photo Access Website" using only **HTML and CSS (no JavaScript functionality required)**.

---

## 🧩 Requirements

### 🔹 Overall Layout

* Create a **dashboard-style layout**
* Include:

  * Left **Sidebar Navigation**
  * Top **Header/Navbar**
  * Main **Content Area**

---

## 📌 Pages / Sections to Design

### 1. 📊 Dashboard Page

* Show summary cards:

  * Total Events
  * Total Photos
  * Total Downloads
* Use card layout with icons
* Clean and modern design

---

### 2. 📁 Manage Events Page

* Table layout with:

  * Event Name
  * Event Code
  * Date
  * Action buttons (View / Delete)
* “Create Event” button at top

---

### 3. 🖼️ Manage Photos Page

* Grid layout of images
* Each photo card contains:

  * Image preview
  * Title
  * Delete button
* Add “Upload Photo” button (UI only)

---

### 4. ⬆️ Upload Page

* Form UI with:

  * File upload input
  * Title input
  * Description input
  * Submit button

---

### 5. 👤 Admin Profile Page

* Profile card:

  * Admin name
  * Email
  * Profile image (dummy)
* Edit button (UI only)

---

## 🎨 UI Design Requirements

* Use **modern and clean design**
* Sidebar:

  * Dark theme
  * Menu items:

    * Dashboard
    * Events
    * Photos
    * Upload
    * Profile
* Use **hover effects** on buttons and cards
* Use **CSS Flexbox / Grid**
* Add **responsive design** (mobile + desktop)

---

## 🎯 Styling Guidelines

* Use consistent spacing and alignment
* Rounded corners (border-radius)
* Soft shadows for cards
* Use color theme:

  * Primary: Blue / Purple
  * Background: Light gray / white
* Buttons:

  * Hover effect
  * Smooth transitions

---

## 📱 Responsiveness

* Sidebar collapses on small screens
* Grid adjusts for mobile view
* Images resize properly

---

## 🚫 Constraints

* Do NOT use JavaScript
* Do NOT connect backend
* Only static UI design

---

## 📝 Expected Output

* Multiple HTML pages OR single page with sections
* Clean, readable CSS
* Professional dashboard look
